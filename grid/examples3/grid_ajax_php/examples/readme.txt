Requeriments:
  - PHP 5 or higher

How to install
  1) Copy files in your www directory
  2) Run /examples/sql/cities.sql in your MySql Admin
  3) Verify the Paths in
        - /examples/config/config.php
        - /examples/freelancesoft/common/ajaxGrid/AjaxGrid.server.php (line 15 only)
        - /examples/xajaxServer/city.server.php (line 15 only)
